QByteArray类的理解与类型转换

2019-05-02

QByteArray 类

QByteArray 类型可简单看作元素是 byte 的数组，可用于存储原始字节（包括 ‘\0’ ）和传统的8位 ‘\0’ 端接字符串。使用 QByteArray 比使用 const char * 更方便。

QByteArray 主要适用于存储原始二进制数据，并且对内存保护要求很高时。（例如，使用嵌入式 Linux 的 Qt ）

QByteArray 的初始化

  1. 在定义时直接赋值，由 const char * 将其传递给其构造函数。
  1. 利用 resize() 函数设置 QByteArray 的大小，并初始化数组中的每个字节。

QByteArray可以嵌入 ‘\0’ 字节, size() 函数总是返回整个数组的大小，包括嵌入的 ‘\0’ 字节，但不包括 QByteArray 添加的终止 ‘\0’。

QByteArray 的常用成员函数

目前主要用到的成员函数如下：

  1. char QByteArray :: at ( int i ) const

    返回索引位置为 $i$ 元素的值，且 $i$ 必须为比特数组中的有效索引值，即 0 <= $i$ < size()。

    如果只需要对 QByteArray 中的元素进行读取，用 at() 函数可以比 operator[] () 更快，因为它不会导致深层拷贝发生。

  2. QByteRef QByteArray::operator[] ( int i )

    返回索引位置为 $i$ 元素的值，且值为可修改的。

  3. char * QByteArray::data ()

    返回指向数组存储数据的指针。该指针可用于访问和修改组成数组的字节。具体访问字节数组中的某一个，采用 ba.data()[0]–> 访问第0个

    得到结果 [H], [e], [l], [l], [o]。

  4. double QByteArray::toDouble(bool *ok = Q_NULLPTR) const

    float QByteArray::toFloat(bool *ok = Q_NULLPTR) const

    int QByteArray::toInt(bool *ok = Q_NULLPTR, int base = 10) const

    返回转换为 double / float / int 类型的字节数组，这里仅为举例说明，几乎所有 QByteArray 转换其他类型均可尝试。
